# Artery Principle

**It's about FLOW, not storage.**

The Artery Principle is NeuralTree's core design philosophy. A well-organized project is not one where files are neatly categorized into folders — it's one where any piece of information can be **found quickly** by an agent or human.

## What It Means

- Organization quality is measured by **retrieval speed**, not folder depth
- A flat structure with good wiring (links, indexes, frontmatter) beats deep nesting with no cross-references
- Every reorganization action must improve information flow, not just tidiness

## How It's Applied

- The [Flow Score](flow-score.md) measures organization quality through retrieval-oriented metrics
- The [0-1-2 Hop Rule](hop-rule.md) enforces maximum retrieval distance
- [Viking Semantic Search](viking-semantic-search.md) provides the retrieval layer
